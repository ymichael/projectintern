# Project Intern Site

Contains information about summer internships, with a focus on NUS SoC.

# Generating Site
- Install [outline-md](https://github.com/ymichael/outline).

```
$ npm install -g outline-md
```

- Run outline to regenerate html files.

```
outline-md
```
